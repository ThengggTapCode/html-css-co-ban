Những điều cơ bản về HTML và CSS trong lập trình web

HTML là gì?

HTML là viết tắt của HyperText Markup Language, dịch ra có nghĩa là Ngôn ngữ Đánh dấu Siêu Văn bản, vì thế nên HTML không phải là ngôn ngữ lập trình.

Trong lập trình web, HTML được sử dụng rộng rãi để tạo nên cấu trúc cơ bản của 1 trang web thông qua các cặp thẻ của nó, trong đó, 1 cặp thẻ sẽ có 1 thẻ đóng và thẻ mở, ví dụ:

<h1>Cặp thẻ tiêu đề</h1>

Trong đó, thẻ mở của dòng code trên là: <h1>, đối với thẻ đóng thì sẽ có 1 dấu gạch chéo đằng trước tên thẻ: </h1>

CSS là gì?

CSS là viết tắt của Cascading Style Sheets, cũng giống như HTML, CSS cũng không phải là Ngôn ngữ Lập trình, CSS chỉ được coi là một Ngôn ngữ Điều chỉnh Giao diện.

Trong Lập trình web, CSS thường được sử dụng để định dạng, tạo kiểu cho các phần tử HTML, giúp trang web trở nên bắt mắt hơn. ! ví dụ cho việc sử dụng CSS trong dự án:

HTML

<body>
    <div class = "box">Box</div>
</body>

CSS

.box {
  font-size: 50px; /* cỡ chữ 50px */
  color: white; /* màu chữ trắng */
  background-color: black; /* nền đen */
  width: 250px; /* rộng 250px */
  height: 250px; /* cao 250px */
}

Output:

Qua CSS, ta sẽ có 1 hình vuông màu đen dài 250px và rộng 250px, bên trong nó có dòng chữ "Box" màu trắng

Cấu trúc 1 file HTML

Dưới đây là cấu trúc cơ bản của 1 file HTML5 (phiên bản mới nhất của HTML):

HTML

<!DOCTYPE html>
<html>
<head>
  <title>Hello World</title>
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>

Trong đó:

<!DOCTYPE html> có tác dụng định nghĩa loại tài liệu HTML đang được sử dụng. Cụ thể, nó cho trình duyệt biết rằng trang web đang sử dụng phiên bản HTML nào để hiển thị nội dung, ví dụ nếu khai báo giống như đoạn code trên, trình duyệt sẽ hiểu đây là tài liệu theo chuẩn HTML5 là phiên bản mới nhất của HTML.

<html></html> là thẻ có vai trò quan trọng, khác với !DOCTYPE html chỉ khai báo cho trình duyệt biết trang web đang sử dụng bản HTML nào, thẻ html sẽ định nghĩa toàn bộ tài liệu và các phần của trang web.

<head></head> thường được sử dụng để chứa các phần thông tin meta về trang web và các tham chiếu đến các tệp tin CSS, JavaScript cũng như những thiết lập khác nhau như tiêu đề trang, tác giả,... Lấy ví dụ từ đoạn code trên, trong thẻ head có 1 thẻ title, khi truy cập vào trang web, trang web sẽ được đặt tên là "Hello World" "Hello World".

<body></body>, tất cả nội dung sẽ phần lớn nằm ở thẻ body, nó chứa toàn bộ nội dung hiển thị của trang web, bao gồm văn bản, hình ảnh, liên kết, biểu mẫu, video, âm thanh và bất kỳ phần nào khác mà người dùng thấy khi họ truy cập vào trang web. Ví dụ như trên, thẻ body chứa 1 thẻ tiêu đề h1 với nội dung"Hello World".

Giới thiệu các thẻ cơ bản của HTML

Dưới đây, mình sẽ liệt kê một số thẻ và phần tử cơ bản để các bạn tham khảo!

Thẻ <h1></h1> đến thẻ <h6></h6>:

Từ <h1></h1> đến <h6></h6>, đây là những thẻ tiêu đề được đánh dấu với kích cỡ từ nhỏ đến lớn, thường được dùng để định dạng các tiêu đề trong trang web, ví dụ:

HTML

<h1>tiêu đề h1</h1>
<h2>tiêu đề h2</h2>
<h3>tiêu đề h3</h3>
<h4>tiêu đề h4</h4>
<h5>tiêu đề h5</h5>
<h6>tiêu đề h6</h6>

Output:

Thẻ <p></p>:

Thẻ <p></p> là thẻ sẽ định dạng văn bản thành các đoạn văn, ví dụ:

HTML

<p>
  Lorem ipsum dolor sit amet consectetur, adipisicing elit.
  Cupiditate reiciendis facere deleniti magnam enim laboriosam
  est neque possimus, asperiores, quas perspiciatis odio aspernatur
  quasi voluptatem repellendus quos cumque aliquam nobis!
</p>

Output:

Thẻ <a></a>:

Thẻ <a></a> sẽ tạo ra các liên kết, khi người dùng nhấn vào sẽ điều hướng đến các trang web khác, ví dụ:

HTML

<a href="https://youtube.com">Youtube</a>
<a href="https://facebook.com">Facebook</a>

Output:

Trong đó:

href = "" là thuộc tính với mục đích xác định đường dẫn đến các liên kết hoặc các dữ liệu trong dự án, ví dụ trên, href sẽ xác định đường dẫn của Youtube và Facebook.

Thẻ <img/>:

Thẻ <img/> là một thẻ tự đóng, khi truyền đường dẫn đến file hình ảnh, hiển thị hình ảnh lên trang web, ví dụ:

HTML

<img src="./assets/img/hello-world.png" 
alt="Hình ảnh dòng code Hello World"/>

Output

Trong đó:

src = "" như thuộc tính href, sẽ xác định đường dẫn, thường là đường dẫn đến các dữ liệu như hình ảnh, v.v../

alt = "" sẽ cung cấp văn bản thay thế cho hình ảnh nếu như hình file hình ảnh gặp trục trặc.

Thẻ <ol></ol>, thẻ <ul></ul> và thẻ <li></li>:

Thẻ <ol></ol> định dạng 1 danh sách có thứ tự, vd: từ 1 đến 5, từ a đến z, ngược lại, thẻ <ul></ul> định dạng 1 danh sách không có thứ tự cụ thể, đối với thẻ <li></li> là thẻ có thể đặt trong cả thẻ ol hoặc ul, có chức năng định dạng các mục trong danh sách, có thể có thứ tự hoặc không, ví dụ:

HTML

<!-- danh sách có thứ tự -->
<ol>
  <li>Hạng 1</li>
  <li>Hạng 2</li>
  <li>Hạng 3</li>
</ol>

<!-- danh sách không có thứ tự -->
<ul>
  <li>Nguyễn Văn A - Bác sĩ</li>
  <li>Lê Thị B - Giáo viên</li>
  <li>Đỗ Hoàng C - Kiến trúc sư</li>
</ul>

Output:

Thẻ <div></div> và thẻ <span></span>:

Thẻ <div></div> và thẻ <span></span> được sử dụng để tạo ra các khối hoặc phần tử nhỏ hơn không chứa bất kỳ dạng định dạng nào mặc định nào, trong đó:

Thẻ <div></div> thường được sử dụng để tạo ra các khối trên trang web, nhóm và tổ chức các phần tử HTML liên quan lại với nhau, tạo layout, cũng như là tạo các khu vực như phần tiêu đề, nội dung chính,... cho trang web.

Thẻ <span></span> được sử dụng để áp dụng kiểu dáng hoặc định dạng cho một phần nhỏ của văn bản, hoặc một phần tử con bên trong một phần tử khác, áp dụng CSS hoặc định dạng nội dung một cách linh hoạt, mà không làm thay đổi cấu trúc của trang web, ví dụ:

HTML

<!-- thẻ div gộp lại thẻ h1, p-->
<div class="header">
  <h1>Đây là Tiêu đề</h1>
  <p>Chú thích cho tiêu đề</p>
</div>

<!-- thẻ span nằm trong thẻ p -->
<p>Đây là thẻ p và thẻ <span>thẻ span</span></p>

CSS

.header {
  font-family: sans-serif; /* font sans serif*/
  font-size: 1.2em; /* cỡ chữ lớn 1.2 lần hiện tại */
}
span {
  font-size: 1.5em; /* cỡ chữ lớn 1.5 lần hiện tại */
  font-weight: bold; /* in đậm */ 
}
